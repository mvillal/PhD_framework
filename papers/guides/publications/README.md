# Publication Guides & Reporting Standards

This directory contains specific guidelines for high-impact venues and international reporting standards for machine learning in clinical research.

## 🏛️ Target Venues
- [[neurips_2026.md]]: Guidelines for the Conference on Neural Information Processing Systems (ML focus).
- [[lancet_psychiatry.md]]: Guidelines for The Lancet Psychiatry (Clinical focus).
- [[nature_medicine.md]]: Guidelines for Nature Medicine (Translational focus).

## 📋 Reporting Standards (The "AI-Extensions")
- **CONSORT-AI:** For randomized controlled trials of AI interventions.
- **SPIRIT-AI:** For clinical trial protocols involving AI.
- **TRIPOD-AI:** For diagnostic and prognostic prediction models using AI.
- **STARD-AI:** For diagnostic accuracy studies using AI.

## 🔄 Usage by Scientific Writer
The `scientific-writer` agent must query this directory when a "Target Venue" is specified in the research task.
