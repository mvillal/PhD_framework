# The AI Scientist

## 📋 Executive Summary
The first comprehensive system for fully automated scientific discovery, taking a research idea from ideation to experimentation and peer-reviewed manuscript.

## 🛠️ Core Methodology
- **Ideation -> Coding -> Experiment -> Review:** A full loop architecture.
- **Template-free Research:** Agents define their own protocols rather than following fixed steps.
