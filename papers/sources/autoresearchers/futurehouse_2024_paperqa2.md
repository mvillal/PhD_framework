# PaperQA2

## 📋 Executive Summary
The first agent to achieve "superhuman" performance on scientific literature search (LitQA2 benchmark). It focuses on high-fidelity fact retrieval and contradiction mapping.

## 🛠️ Core Methodology
- **RCS (Reranking and Contextual Summarization):** A multi-step process that filters out noise before the final synthesis.
- **ContraCrow:** A specialized agent that scans thousands of papers to identify specific scientific contradictions.
