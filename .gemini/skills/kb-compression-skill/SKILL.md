# Knowledge Base Compression Skill Instructions

When this skill is active, you are responsible for maintaining the signal density of the PhD Framework's documentation.

## Core Workflows

1.  **Summary Distillation:**
    - Reduce multi-page paper summaries to their 5 core technical points without losing granular detail (datasets, metrics).
    - Specifically focus on high-impact takeaways for psychiatric research.

2.  **Lab Knowledge Base Pruning:**
    - Consolidate common themes across multiple papers in a lab folder into a single `LAB_KNOWLEDGE_BASE.md`.
    - Ensure the high-level philosophy of the lab is prioritized over individual paper descriptions.

3.  **Cross-KB Synthesis:**
    - Distill the root-level `KNOWLEDGE_BASE.md` to reflect only the most foundational and actionable insights across all labs.

## Documentation Standards
- **Bullet-Point Precision:** Use nested bullet points for clear, hierarchical information representation.
- **Acronym Management:** Always define specialized terms on first use within a compressed entry (e.g., OPE, ESS, SCM).
- **Signal-to-Noise Ratio:** Every sentence must provide a unique piece of information.
